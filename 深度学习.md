# 深度学习优化算法



## 一、反向传播：梯度计算（链式法则应用）

### 核心目标

计算神经网络中**内层参数（如权重$w_j$）的梯度**，为参数更新提供依据（梯度是损失函数对参数的偏导数，反映参数变化对损失的影响程度）。

### 计算逻辑（以权重$w_j$为例）

1. **梯度传递路径**：参数$w_j$的梯度需通过外层参数（如$α_j$）和激活函数$σ$逐层传递，核心是**链式法则**（复合函数求导）。
2. **公式推导**：

$$ \begin{align*} \frac{\partial R*S}{\partial w*j} &= \frac{1}{n}\sum*{i=1}^n (f*θ(x_i) - y_i) \frac{\partial f*θ(x*i)}{\partial w_j} \\ &= \frac{1}{n}\sum*{i=1}^n (f*θ(x_i) - y_i) α_j \frac{\partial σ(w_j·x_i + b_j)}{\partial w_j} \\ &= \frac{1}{n}\sum*{i=1}^n (f*θ(x_i) - y_i) α_j \ σ'(z)\big|*{z=w_j·x_i + b_j} \ · x_i \end{align*} $$ - $R_S$：损失函数（如均方误差）； - $f*θ(x_i) - y_i$：预测值与真实值的误差； - $σ'(z)$：激活函数$σ$在$z=w_j·x_i + b_j$处的导数（引入非线性的关键）； - $x_i$：输入特征（权重$w_j$的梯度与输入直接相关）。

## 二、梯度下降法（Gradient Descent, GD）

### 算法核心

最基础的参数优化算法，**以负梯度方向为搜索方向**（梯度方向是损失函数上升最快的方向，负梯度则是下降最快的方向），通过固定步长（学习率$η$）更新参数，最小化损失函数。

### 算法步骤（第$t$次迭代）

1.  **计算梯度**：在当前参数$θ_{t-1}$下，计算损失函数的梯度： $$g_t = \nabla_θ R_S(θ_{t-1})$$ 2. **更新参数**：沿负梯度方向调整参数： $$θ_t = θ_{t-1} - η·g_t$$ - $η$（学习率）：关键超参数，控制更新幅度（过大会震荡，过小会收敛慢）。

## 三、梯度下降的有效性：泰勒展开解释

### 理论基础

用**一阶泰勒展开**近似参数$θ_{t-1}$邻域内的损失函数： $$R_S(θ) \approx R_S(θ_{t-1}) + (θ - θ_{t-1})^T \nabla_θ R_S(θ_{t-1})$$

### 代入更新规则

将$θ_t = θ_{t-1} - η·\nabla_θ R_S(θ_{t-1})$代入上式： $$ \begin{align*} R*S(θ_t) &\approx R_S(θ*{t-1}) + (-η·\nabla*θ R_S(θ*{t-1}))^T \nabla*θ R_S(θ*{t-1}) \\ &= R*S(θ*{t-1}) - η·\|\nabla*θ R_S(θ*{t-1})\|^2 \end{align*} $$

### 结论

只要梯度不为零（$\nabla R_S \neq 0$）且学习率$η>0$足够小（保证泰勒近似成立），必有： $$R_S(θ_t) < R_S(θ_{t-1})$$ 即**每次更新后损失函数必然下降**，证明梯度下降的有效性。 ## 四、训练稳定性分析（Edge of Stability）

### 简化模型

考虑二次损失函数
$L = \frac{1}{2}λθ^2$，其梯度为$\frac{\partial L}{\partial θ} = λθ$，梯度下降更新规则为： $$θ_{t+1} = θ_t - η·(λθ_t) = (1 - ηλ)θ_t$$

### 稳定收敛条件

为保证参数序列$\{θ_t\}$收敛到 0，需满足线性系统稳定性：更新系数的绝对值小于 1： $$|1 - ηλ| < 1$$

### 学习率范围推导

解不等式得：  
 $$-1 < 1 - ηλ < 1 \implies 0 < ηλ < 2 \implies η < \frac{2}{λ}$$ - $λ$：损失函数 Hessian 矩阵的最大特征值（反映损失函数的曲率）； - 意义：学习率的选择需与损失函数的“陡峭程度”匹配，避免过大导致发散。

## 五、改进优化器：Adadelta 与 RMSProp

### 改进背景

解决**Adagrad 优化器学习率单调递减**的问题（Adagrad 用梯度平方和累积，导致后期学习率过小，难以更新）。

### 核心思想

采用**指数加权移动平均**累积梯度平方，更关注**近期梯度信息**（遗忘远期梯度，避免累积过大）。

### RMSProp 更新规则 1. 计算当前梯度： $$g_t = \nabla_θ R(θ_{t-1})$$

2. 指数加权移动平均累积梯度平方（$v$为衰减率，如 0.99）：

$$n_t = v·n_{t-1} + (1 - v)·g_t^2$$ 3. 更新参数（$\epsilon$为小常数，避免分母为 0）： $$θ_t = θ_{t-1} - \frac{η}{\sqrt{n_t + \epsilon}}·g_t$$

## 六、优化器性能比较（基于 CIFAR-10 数据集）

### 关键结论（从实验结果）

**_SGD_**：训练误差无明确记录，测试误差为 7.65±0.14%；特点是简单稳定，收敛速度较慢，但泛化性能较好。
**HB**：训练误差无明确记录，测试误差为 7.74±0.25%；特点是性能与 SGD 接近，稳定性表现良好。
**RMSProp**：训练误差无明确记录，测试误差 8 为 9.60±0.19%；特点是采用自适应学习率机制，收敛速度较快。
**Adam**：训练误差无明确记录，测试误差为 9.78±0.25%；特点是结合了动量机制与自适应学习率，应用场景广泛，实用性强。 **AdaGrad**：训练误差无明确记录，测试误差为 11.34±0.46%；特点是学习率会单调衰减，后期参数更新速度变慢，难以适配复杂任务。

### 总结

**非自适应方法（如 SGD）**：泛化性能更优，适合对收敛稳定性要求较高的场景，尤其在数据分布简单、需要避免过拟合的任务中表现突出；
**自适应方法**（如 Adam、RMSProp）：收敛速度更快，能快速适配数据特征变化，适合大数据集、复杂模型或需要快速迭代的场景，但部分任务中泛化性能略逊于非自适应方法；

> 实际应用中需根据任务特性选择，小数据集、简单模型可优先选择 SGD；
> 大数据集、复杂模型（如深度学习中的深层网络）可优先选择 Adam 或 RMSProp，平衡收敛速度与训练效果。

## 七、核心公式速记

### 梯度下降更新：

$$θ_t = θ_{t-1} - η·\nabla_θ R_S(θ_{t-1})$$
**RMSProp 更新**：$$θ_t = θ_{t-1} - \frac{η}{\sqrt{n_t + \epsilon}}·g_t$$，
其中 $$n_t = v·n_{t-1} + (1 - v)·g_t^2$$（$v$ 为衰减率，$\epsilon$ 为避免分母为 0 的小常数，$g_t$ 为当前梯度）
稳定学习率范围：$η < \frac{2}{λ}$（$λ$ 为损失函数 Hessian 矩阵的最大特征值，反映损失函数的曲率特性）





# 用于回归的线性神经网络

## 一、核心定位

用于回归的线性神经网络 = 单层全连接神经网络，本质是线性回归的神经网络化，仅含输入层和输出层，无隐藏层，核心用于拟合特征与数值型目标的线性关系，是深度学习入门基础。

## 二、核心结构与符号

### 2.1 网络结构

- 输入层：d个神经元，对应d个特征（仅传数据，无计算）；
- 输出层：1个神经元，输出数值型预测值$\hat{y}$（唯一计算层）。

### 2.2 关键符号

$\boldsymbol{w}$（权重向量）、$b$（偏置）、$\boldsymbol{X}$（设计矩阵，n个样本×d个特征）、$\hat{y}$（预测值）、$y$（真实标签）。

## 三、核心公式（前向传播）

- 向量形式（单样本）：$\hat{y} = \boldsymbol{w}^\top \boldsymbol{x} + b$
- 矩阵形式（批量样本）：$\hat{\boldsymbol{y}} = \boldsymbol{X}\boldsymbol{w} + b$
- 实际模型（含噪声）：$y = \boldsymbol{w}^\top \boldsymbol{x} + b + \epsilon$（$\epsilon$为观测噪声）

## 四、损失函数（量化误差）

核心：平方误差损失（可微，适配梯度下降）

- 单样本：$l^{(i)} = \frac{1}{2}(\hat{y}^{(i)} - y^{(i)})^2$
- 批量损失：$L = \frac{1}{n}\sum_{i=1}^n l^{(i)}$
- 优化目标：找到$\boldsymbol{w}^*, b^* = \arg\min_{\boldsymbol{w}, b} L$

## 五、参数优化（核心方法）

### 5.1 解析解（仅适用于线性模型）

前提：$\boldsymbol{X}$列满秩（$\boldsymbol{X}^\top \boldsymbol{X}$可逆），公式：$\boldsymbol{w}^* = (\boldsymbol{X}^\top \boldsymbol{X})^{-1} \boldsymbol{X}^\top \boldsymbol{y}$

### 5.2 小批量随机梯度下降（SGD，深度学习通用）

- 核心：迭代抽样小批量样本，沿负梯度更新参数
- 关键超参数：小批量大小（32/64等2的幂次）、学习率$\eta$
- 训练流程：初始化参数→抽样批量→计算梯度→更新参数→收敛停止

## 六、关键技巧与理论

- 矢量化加速：用矩阵/向量运算替代for循环，大幅提升训练速度；
- 理论支撑：高斯噪声假设下，最小化平方损失 = 最大似然估计。

## 七、泛化与评估

- 泛化能力：模型在未见过的测试数据上的预测效果；
- 核心指标：均方误差（MSE）、均方根误差（RMSE）、平均绝对误差（MAE）。

## 八、核心小结

线性回归神经网络是单层全连接网络，核心为仿射变换，用平方损失量化误差，通过解析解或小批量SGD优化参数，适用于线性关系的回归任务，是深度学习基础。

# 用于分类的线性神经网络

## 一、核心定位

用于分类的线性神经网络 = 单层全连接神经网络，本质是线性分类器的神经网络化，仅含输入层和输出层，无隐藏层。核心功能是拟合特征与类别标签的线性关系，可处理二分类（k=2）和多分类（k≥3）任务，是深度学习分类任务的入门基础，也是后续复杂分类网络（如CNN、Transformer）的基础组件。

## 二、核心结构与符号

### 2.1 网络结构

- 输入层：d个神经元，对应d个特征（仅传数据，无计算）；
- 输出层：k个神经元（k为类别数，二分类k=2，多分类k≥3），先输出各类别“得分”（未归一化），再通过softmax函数映射为0-1的概率，概率最大的类别即为模型预测类别（唯一计算层）。

### 2.2 关键符号

$\boldsymbol{w}$（权重矩阵，d×k，d为特征数、k为类别数，每一列对应一个类别的特征权重）、$b$（偏置向量，k维，每一个元素对应一个类别的偏移量）、$\boldsymbol{X}$（设计矩阵，n个样本×d个特征，每行对应一个样本的所有特征）、$\hat{\boldsymbol{z}}$（得分向量，k维，未归一化的类别得分）、$\hat{\boldsymbol{y}}$（输出概率向量，k维，元素和为1）、$y$（真实类别标签，二分类可设为0/1标量，多分类常用独热码，仅真实类别对应位置为1，其余为0）。

## 三、核心公式（前向传播）

- 向量形式（单样本）：$\hat{\boldsymbol{z}} = \boldsymbol{w}^\top \boldsymbol{x} + b$（得分向量，$\hat{z}_j$表示第j类的未归一化得分，可正可负），$\hat{\boldsymbol{y}} = \text{softmax}(\hat{\boldsymbol{z}})$（概率向量，将得分归一化，使每个元素处于0-1之间，且所有元素和为1）
- 矩阵形式（批量样本）：$\hat{\boldsymbol{Z}} = \boldsymbol{X}\boldsymbol{w} + b$，$\hat{\boldsymbol{Y}} = \text{softmax}(\hat{\boldsymbol{Z}})$
- softmax函数（单样本第j类概率）：$\hat{y}_j = \frac{\exp(\hat{z}_j)}{\sum_{k=1}^K \exp(\hat{z}_k)}$，核心作用是将任意范围的得分转为0-1概率，总和为1，且得分越高，对应类别的概率越大（解决分类任务的概率输出需求）。

## 四、损失函数（量化误差）

核心：交叉熵损失（适配分类任务，可微且梯度计算简洁，适配梯度下降优化；相比平方损失，能有效解决分类任务中“概率输出与标签不匹配”的问题，避免梯度消失）

- 单样本：$l^{(i)} = -\sum_{j=1}^k y_j^{(i)} \log(\hat{y}_j^{(i)})$（$y_j^{(i)}$为真实标签独热码，仅真实类别对应$y_j^{(i)}=1$，其余为0，因此实际计算仅需关注真实类别对应的预测概率，概率越接近1，损失越小）
- 批量损失：$L = \frac{1}{n}\sum_{i=1}^n l^{(i)}$
- 优化目标：找到$\boldsymbol{w}^*, b^* = \arg\min_{\boldsymbol{w}, b} L$

## 五、参数优化（核心方法）

### 5.1 解析解（极少使用）

因交叉熵损失与softmax函数组合后为非线性函数，仅在极特殊的线性可分且标签分布简单的场景下存在解析解，实际分类任务（尤其是多分类）中几乎不用，优先选择梯度下降类优化方法。

### 5.2 小批量随机梯度下降（SGD，深度学习通用）

- 核心：迭代抽样小批量样本，计算交叉熵损失对参数$\boldsymbol{w}$和$b$的梯度，沿负梯度方向更新参数（softmax与交叉熵组合可简化梯度计算，避免复杂推导，提升训练效率）
- 关键超参数：小批量大小（32/64等2的幂次）、学习率$\eta$
- 训练流程：初始化参数→抽样批量→计算得分与概率→计算梯度→更新参数→收敛停止

## 六、关键技巧与理论

- 矢量化加速：用矩阵/向量运算替代for循环，大幅提升训练速度；
- 理论支撑：softmax+交叉熵组合，等价于对类别标签做最大似然估计，假设真实类别标签服从多项式分布，最大化似然即最小化交叉熵损失，为损失函数的选择提供严格的概率理论依据。

## 七、泛化与评估

- 泛化能力：模型在未见过的测试数据上的分类效果；
- 核心指标：准确率（整体分类正确率，适用于类别均衡场景）、精确率（预测为正类的样本中真实为正类的比例）、召回率（真实为正类的样本中被正确预测的比例）、F1分数（精确率与召回率的调和平均，适用于类别不均衡二分类）、混淆矩阵（多分类专用，直观展示各类别预测与真实标签的匹配情况）。

## 八、核心小结

线性分类神经网络是单层全连接网络，核心为“线性变换（计算类别得分）+softmax映射（转为概率）”，用交叉熵损失量化分类误差，通过小批量SGD优化参数。其适用场景为特征与类别呈线性可分的分类任务，无法处理非线性分类问题，是深度学习分类模型的基础，也是理解复杂分类网络的关键入门模块。
